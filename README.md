<h1>Burger</h1>
<p>A restaurant CLI application that lets the user input the names of burgers they'd like to eat. Uses MySQL as a database and handlebar to serve the html.</p>

<b>Technologies Used: </b>
<ul>
  <li>Node.js</li>
  <li>MySQL</li>
  <li>Express</li>
  <li>Handlebars</li>
  <li>ORM</li>
</ul>
